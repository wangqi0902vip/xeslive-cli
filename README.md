# xeslive-cli

A cli for xeslive to make courseware more convenient.

## Installation

    $ npm install xeslive-cli -g

## How to use?

    $ xeslive init <template> <myProject>
    
## Command
How to find the templates list:

    $ xeslive template
    
How to check the version:

    $ xeslive -v
    
Have fun!
